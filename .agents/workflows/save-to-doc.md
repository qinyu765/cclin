# 保存回答到文档

当用户在学习过程中提问，并希望将回答保存为独立文档时，执行以下步骤：

// turbo-all

## 步骤

1. **确认内容来源**：回顾当前对话中最近一次知识性回答的完整内容。

2. **生成文件名**：根据主题生成简洁的中文文件名，例如 `JSON-Schema详解.md`、`闭包模式.md`、`依赖注入.md`。

3. **写入文档**：将回答内容写入 `docs/<文件名>.md`，格式要求：
   - 开头用一级标题 `#` 概括主题
   - 用 `>` 引用块写一句话摘要
   - 正文保留原始回答的所有代码块、对比表格、解释段落
   - 适当添加 `---` 分隔不同小节
   - **不要**添加原始回答中没有的内容，保持忠实

4. **确认结果**：告知用户文件已创建及路径。
